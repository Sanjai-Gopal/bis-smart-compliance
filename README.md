The **BIS Smart Product Compliance System** is a web-based application that analyzes product descriptions and verifies whether the claims made (such as *shockproof*, *waterproof*, *energy efficient*, etc.) comply with **Bureau of Indian Standards (BIS)** regulations.

The system uses **Artificial Intelligence and Natural Language Processing (NLP)** to automatically extract product claims and map them against predefined BIS standards. It provides compliance status, risk analysis, and consumer-support features through a simple and attractive interface.

This project is developed as part of an **AI & Data Science academic project** and deployed as a **public web application**.
